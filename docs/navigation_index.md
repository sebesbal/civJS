## How To Use This Folder
- Start here when a task is about mechanics or intent rather than current implementation.
- Do not assume every plan/spec detail is fully implemented.
- `spec/` is the most likely place for higher-level requirements.
- `game_concepts/` captures economy/simulation design language.
- `plans/` is planning/history material and is more likely to be stale.
